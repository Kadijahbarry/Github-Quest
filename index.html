CODING QUEST

Welcome to the Coding Quest! Ready to embark on an adventure?

Person A:

You are going to be the leader of this quest, and will be the creator of this website.

Step 1: Create a Repository on GitHub.

Go to GitHub, click the '+' button near the search bar, select 'New Repository.' Make it public, name it, add a README, then go to Collaborators & Teams: Add Collaborators and give your team the admin role.

Person B:

You will be in charge of the HTML file (the backbone of the website). Wait for Person A to make the repository, and you will receive an invite in your GitHub notifications. Click the button below once you are in.

Person C:

You are in charge of the CSS file, the fun styling part! Wait for Person A to make the repository, and you will receive the invite in your GitHub notifications. Click the button below once you are in.

Final Step: Let's deploy!

Go to settings ⇒ Pages ⇒ Choose branch 'main' and folder 'root' then click save. Wait a few seconds for the website to deploy. You will see it on the GitHub code page.
