CODING QUEST

Your mission begins now! Choose your role below to start your journey.

Person A: The Creator

You are the leader of this quest. Your first task is to create the repository and gather your team!

Step 1: Create the Repository on GitHub

Go to GitHub, click the '+' button near the search bar, select 'New Repository'. Make it public, name it, and add a README. Under 'Collaborators & Teams', click 'Add Collaborators' and add your team. Give them the Admin role.

Step 2: Create `script.js`

Copy the JavaScript code below into your new file `script.js`:

document.addEventListener("DOMContentLoaded", () => {
    const revealButton = document.getElementById("revealButton");
    const hiddenMessage = document.getElementById("hiddenMessage");

    revealButton.addEventListener("click", () => {
        hiddenMessage.style.display = "block"; // Show the hidden message
        revealButton.style.display = "none";  // Hide the button after clicking
    });
});

Step 3: Help Fix Mistakes

Oh no, everyone has made mistakes in their code! It's time for you to help each other using GitHub's pull and push feature.
